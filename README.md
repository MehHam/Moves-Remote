# Moves Remote :

A simple Unity UPD remote to use a smartphone as a gesture remote. UDP messages are of type " "Acc\t" + Input.acceleration.ToString() +"\t"+ "Gyro\t" + gyro.gravity.ToString()" data can be split using String.split('\t');
